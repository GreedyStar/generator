
# 欢迎来到 Generator

Generator 是一款基于数据库表生成相应Java代码的工具，代码模板使用当前主流Java框架： **Spring, SpringMVC, Mybatis** 组织，能够减少繁琐的重复性工作，让开发人员更专注于技术和性能，提高工作效率和编码热情。

你可以使用Generator：
> * 根据数据库业务表生成实体类
> * 生成包含简单的增、删、查、改操作的Mapper文件
> * 生成Controller、Service、Dao代码
